## Requirements

### Setup

<pre>$ npm install</pre>

### External repository

* Compiled TON token file in plasma-evm-contracts

<pre>$ cd plasma-evm-contracts && truffle compile && cd -</pre>

## Testing

<pre>$ npm run-script test_daemon</pre>
